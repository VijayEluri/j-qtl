Genotype Plot

The Genotype Plot can be used to visualize genotype, crossover, missing data or likely genotype errors.

 

Genotype Plot Settings:

The genotype plot settings are integrated into the graph itself. In order to see the effect of a settings change you need to click the "Redraw" button. In order to render rows for selected individuals, enter a comma separated list of individual numbers into the "Choose Individuals" text field.
