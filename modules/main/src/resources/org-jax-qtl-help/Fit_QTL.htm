Fit QTL

The Fit QTL dialog acts as a graphical interface to R/qtl's makeqtl and fitqtl functions. See the R/qtl documentation for detailed information on how these functions work.

 

Fit Cross, Phenotype and QTL Basket Selection:

The first three drop-down selection boxes allow the user to select the cross and phenotype that the fit will be performed against along with the QTL Basket that will be the source of all available marker model inputs.

 

Fit Formula Editing:

The next section of the dialog contains tables and buttons that can be used to edit the fit formula. The model inputs table contains all of the available inputs to the formula, and the model terms which is initially empty contains a list representation of the terms selected. The Formula Editing buttons work as follows:

Button Name Button is Enabled When ... Button Action
Add Term(s) Any Model Inputs are selected Adds the selected inputs as individual additive terms
Add as Interactive Term More than one Model Input is selected Adds the selected inputs as an interactive term. Since J/qtl uses the '*' operator instead of the ':' operator for interaction, the terms are also treaded as additive.
Append Interaction(s) to Term At least one Model Input is selected and one Model Term is selected Appends the selected Model Input as an interaction with the selected model term.
Remove Term A Model Term is selected Removes the selected term from the model's formula

 

Drop-One-Term, Estimate QTL Effects and Fit Analysis Name:

The remaining settings work as follows:

  • Drop-One-Term Analysis: performs additional analysis on the Fit Model which can be helpful in determining which model terms are the most/least significant. If you choose to perform Drop-One-Term Analysis, the results will be available in the Fit Results dialog.
  • Estimate QTL Effects: performs additional analysis output which is currently not used by J/qtl, but which can be acessed directly from R/qtl. See the fitqtl documentation for details.
  • Name Your Fit Analysis: Determines which name the Fit QTL results will be given in the J/qtl data tree
