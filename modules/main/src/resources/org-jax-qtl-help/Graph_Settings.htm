Graph Settings

Most of the graphs with an X and Y axis in J/qtl have a tool bar that allows you to edit graph settings or export a graph to image format:

If you select the settings icon you will be shown a dialog where you can edit the graph axis labels or the graph title. Depending on what graph type you are configuring you may also see additional controls for zooming, panning (this can be done by changing the axis origin and extent values), fonts etc...

Selecting the image icon allows you to export the graph image to PNG format (compatible with most browsers, document editors and image viewers).
