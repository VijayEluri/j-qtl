Edit QTL Basket

The QTL Basket can be used to store the position and user notes for a crosses' candidate QTLs.

For each cross there can be zero to many QTL Baskets and for each QTL Basket there can be zero to many candidate QTLs. The "View/Edit QTL Basket" dialog allows the user to

  • Add new Candidate QTLs via the "Add QTL ..." button (see also One QTL Genome Scan Summary and Two QTL Genome Scan Summary for an alternate way to add QTLs). Upon clicking the "Add QTL ..." button the user will be asked to select from a list of available markers:
  • Remove QTLs by selecting the table rows for the QLTs that should be removed, and clicking "Remove Selected QTL(s)" button.
  • Export the candidate QTL table to a comma-separated file via the "Export Table ..." button

Once there are QTLs in the QTL Basket, they can be used as either model terms for Fit QTL or as covariates in the One QTL Genome Scan and Two QTL Genome Scan dialogs.
