One QTL Genome Scan Summary

The One QTL Genome Scan Summary or "Scan One Summary" dialog can be used to view and filter the output of a One QTL Genome Scan. In effect, this dialog is acting as a graphical interface for R/qtl's summary.scanone function. See the R/qtl documentation for more detailed information on how the scan results are filtered.

 

The initial set of three drop-down selection controls can be used to select the cross, scan results and phenotype to summarize. The "Threshold" dropdown can be used to filter the results by LOD score or by an alpha threshold (the alpha threshold is only available if permutations have been calculated).

The control buttons at the bottom of the dialog work as follows:

  • Toggle Select All Displayed: will either select or deselect all table rows that have not been filtered out
  • Add Selected Markers to QTL Basket: will add the markers from all of the table rows that have checked boxes to a QTL Basket.
  • Export Table: allows the user to export the table to a comma-separated file
