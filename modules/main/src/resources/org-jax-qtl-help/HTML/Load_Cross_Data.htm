Load Cross Data

The "Load Cross Data" dialog can be used to load QTL experiment data that has been stored in either a Comma-Delimited or a Rotated Comma-Delimited file format. This dialog is essentially a graphical interface for the R/qtl read.cross function. See the R/qtl documentation for read.cross for detailed information on the file formats that can be read and the different parameters that can be set.

After a valid "Cross Data File" is selected, the dialog will display a preview of the data in the file in order to make determining cross type and genotype codes easier:

If the "Load Cross" button is clicked, the command in the preview panel will be executed and the data tree in the main window will display the newly loaded cross data.
