Installation and Uninstallation

Installing J/qtl:

The following dependencies should be installed on your computer before attempting to run J/qtl:

Hard Requirements (application will not start without these):

  • Windows XP or Mac OS X >= 10.4
  • Java Runtime >= 5.0

Soft Requirements (the user will be prompted if they don't meet these requirements):

  • R >= 2.6
  • R/qtl >= 1.08

After the software dependencies have been installed, you can visit http://research.jax.org/faculty/churchill/software/Jqtl and click on the install link to begin downloading and installing J/qtl. During the installation process the user will be prompted to select an R installation to run against. After selecting an R installation J/qtl will remember this decision. The selected installation can be changed at any time using the edit preferences dialog.

Uninstalling J/qtl on Windows XP:

In order to uninstall J/qtl on Windows XP, you can go to Start->Control Panel->Add or Remove Programs, select J-QTL and click the "Remove" button. If you want to remove any remaining J/qtl configuration files you can manually delete the C:\Documents and Settings\<user name\.j-qtl directory.

Uninstalling J/qtl on Mac OS X:

For Mac OS X, open Applications->Utilities->Java->Java Web Start. Once the Java Web Start utility is open select the J-QTL application and click the "Remove Selected Application" button. If you want to remove any remaining J/qtl configuration files you can delete the <user home>/.j-qtl directory.
