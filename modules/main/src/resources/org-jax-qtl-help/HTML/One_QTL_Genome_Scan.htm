One QTL Genome Scan

The "One QTL Genome Scan" dialog is a graphical interface for the R/qtl scanone function. See the scanone documentation for detailed information on how each of the parameters affect the outcome of the scan.

 

Initial Panel Scan Setting Panel

The initial scan panel allows the user to select:

  • The cross to scan
  • The phenotype to scan
  • The phenotype distribution that the scan assumes

 

Chromosomes and Covariate Selection Panel

The upper section of this panel can be used to select the chromosomes that should be scanned and the lower section allows you to select which covariates will be used. Covariates can either be phenotypes or marker covariates (marker covariates will only be available if markers have already been added to a QTL Basket ). Note that any Interactive covariates are also forced to be Additive covariates.

 

Scan Method Panel

The Scan Method Panel allows the user to select the Scan Method to use and convergence parameters for the EM Algorithm and Extended Haley Knott scan method.

 

Scan Name and Permutations Panel

The final panel is used to set the name of the scan result and permutation settings. Performing permutations can be computationally expensive, but they are useful for determining alpha thresholds for both the One QTL Scan Plot and the One QTL Summary filters.
