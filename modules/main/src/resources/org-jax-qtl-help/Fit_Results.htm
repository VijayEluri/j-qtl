Fit Results

The Fit Results dialog can be used to inspect the output of Fit QTL. See the R/qtl documentation for fitqtl to get detailed information on this output.

The Fit Results panel allows to select which Cross and Fit Results you are viewing data for. In addition to the Cross and Fit Results selection controls you will see either one or two tabbed panels depending on whether or not you chose to perform "Drop-One-Term" analysis on the Fit Qtl model. There is also an "Export Table ..." button at the bottom of the dialog that can be used to export either the Full Results table or the Drop-One-Term Results table to comma-seperated format (whichever table is visible gets exported).

 

Full Results Tab:

The Full Results tab shows the output of doing a fit on the full model (ie: no dropped terms).

 

Drop-One-Term Results Tab:

This tab will only be visible if the "Drop-One-Term" checkbox was selected for Fit QTL.
