Two QTL Genome Scan Summary

The "Two QTL Genome Scan Summary" is used to summarize the result of a Two QTL Genome Scan . This dialog acts as a graphical interface to R/qtl's summary.scantwo function. See the R/qtl documentation for detailed information on how this function works. Please pay special attention to the description of how the threshold filter works since this might not be obvious.

The marker pair names can be clicked to see a Jackson Laboratory - Mouse Genome Informatics (http://www.informatics.jax.org) web page that gives information about the region of the chromosome where the markers can be found.

If the scan was calculated using permutations, you will also be able to set alpha values for the threshold filter. The p-values calculated will also be shown in the table if the Model to Optimize is set to "Full and Additive Models". P-values will not be shown for any other models even if permutations are calculated.

The action buttons work as follows:

  • Toggle Select All Displayed: toggles the checkmarks between all selected and all deselected
  • Add Selected Markers to QTL Basket: adds the marker pairs where the selected checkbox is checked to a QTL Basket
  • Export Table: exports the table to a comma-separated file
