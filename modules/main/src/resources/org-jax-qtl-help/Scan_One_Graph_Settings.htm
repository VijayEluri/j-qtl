Scan One Graph Settings

In addition to the graph label available for on other graphs (see Graph Settings) the scan one graph settings allow you to modify the graph intervals and thresholds.

 

Intervals

The intervals panel allows you to select either LOD Drop intervals or Bayesian Credible intervals and choose which chromosomes the intervals will show up for. J/qtl uses the R/qtl functions bayesint and lodint to calculate these intervals. Clicking "Print Interval(s) to Terminal" will cause the R/qtl commands to be executed in the terminal.

 

Thresholds

The thresholds panel allows you to add or remove alpha threshold values to the scanone graph. Thresholds are calculated using the summary.scanoneperm R/qtl function. This panel will only be available if permutations were calculated during the scan.
